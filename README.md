### Documentation Libraries ###
* Socket.io - [https://socket.io/docs/]
* Threejs - [https://threejs.org/docs/]

### Build Project ###
* Install
```
git clone git@github.com:BasicMan01/webgl_metaverse.git webgl_metaverse
cd webgl_metaverse
npm install
```

* Build Client
```
npm run prod
```

### Start Server ###
```
node src/server/server
```
